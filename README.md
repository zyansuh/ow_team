# 티어맞춤 팀짜기

오버워치 커스텀·서버 내전을 위한 **역할별 티어 밸런스 팀 편성** 웹앱입니다.  
닉네임과 **포지션별 티어**를 등록하면 **탱커 1 · 딜러 2 · 힐러 2** 구성으로 팀을 나누고, 원하는 팀 수만큼 싱글 엘리미네이션 토너먼트까지 진행할 수 있습니다.

> 대규모 서버(수백 명) 내전을 염두에 두고, **팀원·팀 수 모두 제한 없이** 늘릴 수 있습니다.  
> **탱·힐·딜 티어를 모두 적은 뒤 무작위(플렉스)** 를 켜면, 팀짜기 때 **빈 슬롯을 해당 포지션 티어로 자동 채웁니다.**  
> **미배치/언랭**과 **모바일·태블릿·데스크톱** 반응형을 지원합니다.

**저장소:** [https://github.com/zyansuh/ow_team](https://github.com/zyansuh/ow_team)

## 주요 기능

| 기능 | 설명 |
|------|------|
| OW 5인 구성 | 팀당 **탱커 1 · 딜러 2 · 힐러 2** (완편 여부 표시) |
| 팀원 등록 | 닉네임 + 포지션별 티어 · 인원 제한 없음 |
| 멀티 포지션 티어 | 탱/힐/딜마다 다른 티어 입력 (예: 탱 골드3, 힐 플래2) |
| 무작위(플렉스) | 탱·힐·딜 티어를 모두 작성한 뒤 켜면 **빈 슬롯 자동 투입** |
| 티어 체계 | **미배치/언랭** + 브론즈~챔피언 × 디비전 1~5 |
| 포지션별 밸런싱 | 전담 먼저 배치 → 플렉스가 빈자리 채움 → 같은 슬롯끼리 MMR 미세 조정 |
| 팀 수 | 1팀부터 무제한 (`+/-`, 직접 입력, `+5` / `+10` / `+20`) |
| 토너먼트 | N팀 싱글 엘리미네이션 · 부전승 자동 · 가로 스냅 스크롤 |
| 로컬 저장 | `localStorage` · 구버전 데이터 자동 마이그레이션 |
| 반응형 UI | 풀폭 버튼, 터치 영역, 모바일 브래킷 스와이프 |

## 팀짜기 흐름 (요약)

1. **전담** (무작위가 아닌 인원)을 역할 슬롯에 티어 맞춰 배치  
2. **무작위(플렉스)** 인원을 **빈 슬롯**(부족한 탱/딜/힐)에 해당 포지션 티어로 채움  
3. 남는 인원은 인원·MMR이 적은 팀에 오버플로 배치  
4. 같은 배정 역할끼리 스왑해 팀 간 역할 MMR 편차를 줄임  

완편 기준: 팀당 `탱 1/1 · 딜 2/2 · 힐 2/2` (인원이 부족하면 일부 팀만 완편).

### 알고리즘 (직접 구현)

- **OW 5인 슬롯 정원** — `TEAM_COMPOSITION = { tank: 1, dealer: 2, healer: 2 }`
- **전담 우선 배치** — 역할 티어 기준으로 슬롯별 균형 채우기
- **플렉스 빈 슬롯 투입** — 무작위 인원을 부족한 탱/딜/힐에 해당 티어로 배치
- **배정 역할(slottedRole) 스왑** — 같은 슬롯끼리 MMR 편차 최소화
- **N팀 브래킷** — 2의 거듭제곱 패딩 · 부전승 · 승자 다음 라운드 연결

백엔드·DB 없음. 팀원 데이터는 브라우저 `localStorage`만 사용합니다.

## 데이터 모델 (요약)

```ts
Player {
  id, nickname,
  roles: [
    { position: 'tank'|'healer'|'dealer'|'random', tier: { rank, division } },
    ...
  ]
}

TeamMember {
  player: Player
  slottedRole: 'tank'|'healer'|'dealer'  // 이 팀에서 맡은 역할
}

Team {
  id, name,
  members: TeamMember[],
  totalMmr  // 배정 역할 티어 MMR 합
}
```

- **무작위**: 별도 티어 없음. 탱·힐·딜 티어를 모두 넣은 뒤 플렉스 표시로 사용
- **역할 평균 MMR**: `slottedRole`이 그 역할인 멤버만, **그 포지션 티어**로 집계
- **미배치/언랭**: `rank: 'unranked'`, MMR `0`, 디비전 없음

## 사용 방법

1. **로비**에서 닉네임 입력  
2. **탱커·힐러·딜러** 티어를 각각 작성 (`미배치/언랭` 가능)  
3. 플렉스로 돌릴 사람이면 **무작위** 켜기  
   - 켜는 순간 탱·힐·딜이 모두 켜지며, **세 티어를 다 적어야** 등록 가능  
4. **팀 수** 설정 (예: 2팀 스크림, 팀당 5명이면 완편)  
5. **티어 맞춰 팀짜기**  
6. 결과에서 **배정 포지션**, 구성 `탱 n/1 · 딜 n/2 · 힐 n/2`, 완편 여부 확인  
7. 토너먼트에서 승자 선택 (모바일은 브래킷을 좌우로 밀기)

최소 인원: **팀 수 이상**.  
5인 완편을 모두 채우려면 대략 **팀 수 × 5** 명이 필요합니다.

## 티어 MMR (요약)

낮음 → 높음:

`미배치/언랭` → `브론즈 5` … `브론즈 1` → `실버` … → `챔피언 1`

| 항목 | 내용 |
|------|------|
| 미배치/언랭 | 배치 전·언랭크. MMR `0`, 디비전 없음 |
| 디비전 | 5(낮음) → 1(높음) |
| 점수 | `rankIndex * 5 + (6 - division)` (미배치는 `0`) |
| 팀 합계 | 멤버별 **배정된 역할**의 티어 MMR 합 |
